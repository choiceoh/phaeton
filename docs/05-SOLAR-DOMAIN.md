# 한국 태양광 발전사업 — 인허가 및 공사 도메인 가이드

> Phaeton ERP에서 태양광 프로젝트의 인허가 추적, 마일스톤 관리, 컴플라이언스 관리를 위한 도메인 참조 문서.
> 2025년 기준 시행 법령 및 제도 기반. 법규 변동이 잦으므로 최신 확인 필요.

---

## 1. 태양광 프로젝트 전체 흐름

```
(1) 사업개발/기획 ─── planning
 └→ (2) 발전사업허가 ─── permit
     ├→ (3) 개발행위허가
     ├→ (3-1) 농지전용허가 (농지인 경우)
     ├→ (3-2) 산지전용허가 (산지인 경우)
     ├→ (3-3) 소규모환경영향평가 (해당 시)
     └→ (4) 공사계획 인가/신고
         └→ (5) 착공 ─── construction
             └→ (6) 토목 → 구조물 → 모듈 → 전기공사
                 └→ (7) 시운전/사용전검사 ─── testing
                     └→ (8) 계통연계 + PPA 체결
                         └→ (9) 상업운전개시(COD) ─── cod
```

**Phaeton 프로젝트 상태 매핑:**
| 흐름 단계 | Projects.status |
|-----------|----------------|
| 사업개발/기획 | `planning` (기획) |
| 인허가/행정 | `permit` (인허가) |
| 시공 | `construction` (시공) |
| 시운전/검사 | `testing` (시운전) |
| 상업운전 | `cod` (운영) |

## 2. 인허가 절차 상세

### 2-1. 발전사업허가

| 항목      | 내용                                                    |
| --------- | ------------------------------------------------------- |
| 근거 법률 | 전기사업법 제7조                                        |
| 관할 기관 | 3MW 이하: 시장/군수/구청장, 3MW 초과: 시/도지사         |
| 소요 기간 | 50~90일                                                 |
| 선행 조건 | 부지 확보(소유 또는 사용 동의), 한전 계통 용량 사전확인 |

**주요 서류:**

- 발전사업허가 신청서, 사업계획서
- 공사비 계산서, 발전원가 명세서 (200kW 이상)
- 기술인력 확보계획 (200kW 이상)
- 발전설비 개요, 전기설비 설치일정
- 송전관계 일람도, 모듈 배치도/측면도
- 토지사용동의서, 지적도, 등기부등본

**규모별 차이:**

- 200kW 미만: 발전원가 명세서, 기술인력 확보계획 면제
- 1MW 이하: 개발행위허가/농지전용허가 의제처리 가능 (2025년~)
- 허가 후 **2년 내 미착공 시 허가 취소**

---

### 2-2. 개발행위허가

| 항목      | 내용                                                 |
| --------- | ---------------------------------------------------- |
| 근거 법률 | 국토의 계획 및 이용에 관한 법률 제56조               |
| 관할 기관 | 시장/군수/구청장                                     |
| 소요 기간 | 15~120일 (주민 설명회 포함 시 장기화)                |
| 선행 조건 | 발전사업허가 (또는 동시 진행), 부지 소유/사용권 확보 |

**주요 서류:**

- 개발행위허가 신청서, 토지 소유권/사용권 증빙
- 배치도, 설계도서, 경관/조경 설계도서
- 예산 내역서, 배수계획서

**핵심 주의:**

- 인허가 중 **가장 변수가 많은 단계**
- 1만 m2 이상 시 주변 500m 이내 **주민 설명회 의무** (지자체 조례)
- 지자체별 이격거리 조례가 상이
- 발전사업허가 취득이 개발행위허가를 보장하지 않음

| 관할 기관 | 시장/군수/구청장                    |
| 소요 기간 | 30~60일                             |
| 적용 조건 | 부지가 농지(전, 답, 과수원)인 경우  |
| 비용      | 개별공시지가의 30% — 농지보전부담금 |

**영농형 태양광:** 농업 활동과 병행하는 특례 (농지의 타용도 일시사용허가). 농업진흥지역은 원칙적 불가, 영농형 특례로 일부 가능.

## 4. 공사 단계 상세

### 4-1. 사업개발/기획

**세부 작업:**

1. 후보 부지 조사 및 선정
2. 일사량/일조시간 분석 (GIS 기반 잠재량 분석)
3. 계통연계 여유용량 확인
4. 사업성 검토 (LCOE, IRR, NPV 분석)
5. 토지 매매/임대 계약
6. 사업계획서 수립
7. 자금조달 계획 (PF, 자기자본)

**산출물:** 사업타당성 분석 보고서, 토지 계약서, 재무모델

**소요기간:** 3~12개월

---

### 4-2. 설계

#### 기본설계 (Basic Design)

- 배치설계: 모듈 배치, 이격거리, 방위각/경사각 최적화
- 시스템 용량 설계: 모듈, 인버터, 접속반 용량 결정
- 단선결선도 작성
- 발전량 시뮬레이션 (PVsyst 등)

#### 실시설계 (Detailed Design)

- 토목설계: 부지 조성, 배수계획, 진입로, 기초설계
- 구조설계: 하중계산, 풍하중/설하중/지진하중 검토
- 전기설계: 케이블 사이징, 접지설계, 보호협조, 수배전 설비

**산출물:** 설계도서 (토목, 구조, 전기), 구조계산서, 발전량 예측 보고서

**소요기간:** 1~3개월

---

### 4-3. 조달 (Procurement)

**세부 작업:**

1. EPC 계약 체결 (또는 분리발주 시 시공사 선정)
2. 주요 기자재 발주
3. KS 인증품 확인 (모듈, 인버터, 접속함 — KS 인증 의무)
4. 자재 납기 관리 및 입고 검수

**산출물:** 기자재 발주서, KS 인증서, 자재 검수 보고서, 시험성적서

**소요기간:** 1~3개월 (모듈 리드타임에 따라 변동)

---

### 4-4. 착공 준비

**세부 작업:**

1. 공사계획신고 (전기사업법 제62조) — 착공 15일 전
2. 전기 감리자 선정 및 배치
3. 안전관리계획서 작성 (건설기술진흥법)
4. 유해위험방지계획서 제출 (해당 시, 산업안전보건법)
5. 착공신고서 제출

---

### 4-5. 토목공사

**세부 작업:**

1. 현장 경계 측량 및 먹줄 작업
2. 벌개제근 (산지의 경우)
3. 부지 정지 공사 (절토/성토)
4. 배수로 공사 (우수 배수, 유수지)
5. 진입도로 공사
6. 울타리/안전펜스 기초
7. 기초공사
   - 파일기초: 스파이럴, 스크류, 레이밍 파일, 보링그라우팅 공법
   - 콘크리트 기초: 독립기초, 줄기초
8. 가설 전기/용수 확보

**소요기간:** 1~3개월

---

### 4-6. 구조물 설치

**세부 작업:**

1. 컬럼베이스 + 브라켓 볼팅
2. 포스트(기둥) 세우기
3. 거더(대들보) 조립
4. 퍼린(중도리) 설치
5. 보강대(브레이싱) 설치
6. 수평/수직 정밀도 확인

**구조물 유형:**

- 재질: 용융아연도금 스틸, 알루미늄
- 방식: 고정식(Fixed-tilt), 단축 추적식(Single-axis tracker), 양축 추적식(Dual-axis tracker)

**소요기간:** 1MW 기준 약 1~2주

---

### 4-7. 모듈 설치

**세부 작업:**

1. 모듈 현장 반입 및 검수
2. 모듈 양중 및 배치
3. 모듈 고정 (클램프 체결)
4. 모듈 간 직렬 결선 (스트링 연결)
5. 스트링별 개방전압(Voc) 측정
6. 모듈 외관 검사

**산출물:** 모듈 설치 대장 (시리얼넘버), 개방전압 측정표

**소요기간:** 1MW 기준 약 1~2주

---

### 4-8. 전기공사

**세부 작업:**

1. 접속반(접속함) 설치 — SPD, 역전류 방지 다이오드
2. DC 케이블 포설 (모듈~접속반~인버터)
3. 인버터 설치 (스트링 또는 센트럴)
4. 변압기/수배전설비 설치 (500kW 이상)
5. AC 간선/인입 공사
6. 접지공사 — 접지극 매설, 접지저항 측정
7. 피뢰설비 설치 (해당 시)
8. 계통연계설비 설치 — 보호계전기(OVR, UVR, OFR, UFR) 설정
9. CCTV/보안 설비 설치
10. 모니터링 시스템 설치
11. 울타리(안전펜스) 설치

**산출물:** 시공도면(준공도), 절연저항 측정표, 접지저항 측정표, 보호계전기 정정값 기록

**소요기간:** 1MW 기준 약 2~4주

---

### 4-9. 시운전 및 검사

**자체 시운전:**

- 절연저항/접지저항 측정 (DC/AC 회로)
- 인버터 단독 시운전
- 보호계전기 동작 시험
- 스트링별 I-V 특성 검사, 출력 확인

**공적 검사:**

1. **사용전검사** (KESCO) — 전기설비기술기준 적합 여부
2. **계통연계 시험** (한전) — 병렬운전, 단독운전방지, 보호협조
3. **준공검사** (지자체) — 개발행위허가에 따른 준공
4. **RPS 설비 설치확인** (에너지공단) — REC 발급 대상 확인

**소요기간:** 1~2개월

---

### 4-10. 상업운전개시 (COD)

1. 전력수급계약(PPA) 체결
2. 사업개시신고 (관할 지자체)
3. SMP 정산 및 REC 거래 개시
4. 발전소 인수인계 (EPC → 사업주)
5. 유지보수(O&M) 계약 체결

### 주요 제도

#### RPS (신재생에너지 공급의무화제도)

- 500MW 이상 발전사업자(의무공급자)에게 일정 비율의 신재생에너지 공급 의무
- 2026년 기준 약 13% 내외
- 미이행 시 REC 평균거래가격의 150% 이내 과징금

#### FIT (고정가격계약 입찰)

- 대상: 30kW 이상 ~ 1MW 미만 소규모 태양광
- 한국에너지공단 주관 입찰, **20년 고정가격** 계약 (SMP+REC 고정)

#### 한국형 RE100 (K-RE100)

- 전기 사용량 연 1GWh 이상 기업 자발적 참여
- 이행 수단: 녹색프리미엄, REC 구매, 제3자 PPA, 자가발전, 지분투자

#### 직접 PPA (제3자 PPA)

- 2021년부터 시행, 발전사업자와 전기사용자 간 직접 계약
- 통상 15~20년 장기계약
- 한전 계통 이용료(송배전 이용요금) 별도
