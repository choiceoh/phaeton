# Topworks (Phaeton) — 구현 기능 목록

> 최종 갱신: 2026-04-11

---

## 1. 인증 & 사용자 관리

### 1.1 인증
- 이메일/비밀번호 로그인 (JWT, 7일 만료, httpOnly 쿠키 + Bearer 헤더)
- 로그아웃 (쿠키 삭제)
- SAML 2.0 SSO (crewjam/saml 기반, IdP 메타데이터 URL 연동)
- 비밀번호 변경 (현재 비밀번호 검증 필수)
- 로그인 실패 잠금 (IP당 15분 내 5회 실패 → 30분 차단)

### 1.2 사용자 관리 (관리자 전용)
- 사용자 목록 조회 (부서/자회사 필터)
- 사용자 생성·수정·비활성화
- 역할 지정: director, pm, engineer, viewer
- 부서·자회사 배정

### 1.3 프로필
- 이름, 전화번호, 직위, 직책, 아바타 수정
- 알림 설정 (데스크탑 알림, 소리 알림 토글)

## 2. 조직 구조

### 2.1 자회사 (Subsidiary)
- CRUD (생성, 조회, 수정, 삭제)
- director만 쓰기 권한

### 2.2 부서 (Department)
- 트리 구조 지원 (`?tree=true`)
- 상위 부서 계층 관리
- 순환 참조 방지
- director만 쓰기 권한

### 2.3 조직도
- 자회사 → 부서 → 사용자 트리 시각화
- 사용자 상세 정보 패널 (이니셜 아바타, 소속, 연락처)

## 3. 앱(컬렉션) 관리

### 3.1 앱 생성
- 이름, 슬러그, 설명, 아이콘 설정
- AI 기반 앱 자동 생성 (자연어 설명 → 스키마)
- AI 슬러그 생성
- 템플릿 갤러리에서 원클릭 생성
- 템플릿 내보내기/가져오기 (JSON: 컬렉션 + 필드 + 뷰 + 자동화 + 차트)

### 3.2 앱 빌더 (3-패널 UI)
- 드래그앤드롭 필드 팔레트
- 실시간 폼 미리보기
- 필드 속성 편집 (라벨, 타입, 옵션, 유효성 검사)

### 3.3 앱 설정
- 이름, 슬러그, 설명, 아이콘 변경
- 컬렉션 멤버 관리 (추가, 역할 변경, 제거)
- 행 수준 보안(RLS) 필터 설정
- 앱 삭제 (확인 필수)

### 3.4 앱 목록
- 전체 앱 탐색 및 검색
- 앱별 레코드 수 표시
- 온보딩 가이드 (신규 사용자용)
- 관계도 링크

| `multiselect` | 다중 선택 |
| `relation` | 관계 필드 (1:1, 1:N, M:N) |
| `user` | 사용자 참조 |
| `file` | 파일 첨부 |
| `json` | JSON 데이터 |

| 타입 | 설명 |
|------|------|
| `formula` | 수식 (SQL 표현식으로 변환) |
| `lookup` | 관계 레코드의 특정 필드 값 조회 |
| `rollup` | 관계 레코드 집계 (SUM/COUNT/AVG/MIN/MAX) |

### 4.3 레이아웃 필드 (DB 컬럼 없음, 폼 UI 전용)
| 타입 | 설명 |
|------|------|
| `label` | 텍스트 라벨 |
| `line` | 구분선 |
| `spacer` | 여백 |

### 4.4 자동 주입 컬럼 (모든 동적 테이블)
- `id` (UUID, PK, 자동생성)
- `created_at`, `updated_at` (TIMESTAMPTZ)
- `created_by`, `updated_by` (UUID, 사용자 참조)
- `deleted_at` (소프트 삭제)
- `_status` (프로세스 워크플로 상태)

## 5. 수식 엔진

### 5.1 지원 함수
- **집계**: SUM, AVG, MIN, MAX, COUNT
- **스칼라**: ROUND, CEIL, FLOOR, ABS
- **논리**: IF, COALESCE, NULLIF
- **관계 집계**: LOOKUP, SUMREL, AVGREL, MINREL, MAXREL, COUNTREL

### 5.2 연산자
- 산술: `+`, `-`, `*`, `/`, `%`
- 비교: `<`, `>`, `<=`, `>=`, `=`, `!=`, `<>`
- 논리: `AND`, `OR`

### 5.3 특징
- 3단계 파이프라인: 렉서 → 파서 → PostgreSQL SQL 생성
- 파라미터 바인딩으로 SQL 인젝션 방지
- 필드 슬러그 유효성 검증
- 수식 미리보기 API

## 6. 데이터 뷰

### 6.1 리스트(테이블) 뷰
- 인라인 셀 편집 (낙관적 업데이트)
- 컬럼 정렬 (다중 컬럼, ASC/DESC)
- 컬럼 리사이징/재배치
- 행 선택 (다중 선택 → 벌크 작업)
- 서버사이드 페이지네이션 (page/limit)

### 6.2 칸반 뷰
- 선택 필드 기준 컬럼 그룹핑
- 드래그앤드롭 카드 이동
- 컬럼 접기/펼치기
- 컬럼별 항목 추가 버튼
- 키보드 내비게이션

### 6.3 캘린더 뷰
- 월/주/일 뷰 전환
- 날짜 필드 기반 이벤트 표시
- 이벤트 클릭 → 상세 열기
- 다중 컬렉션 이벤트 (색상 코딩)

### 6.4 갤러리 뷰
- 이미지 그리드 표시
- 이미지 지연 로딩 + 에러 핸들링
- 카드 상세 필드 표시

### 6.5 간트 차트 뷰
- 타임라인 바 시각화
- 바 드래그로 날짜 범위 편집
- 담당자/상태별 색상 코딩
- 항목 목록 패널 (접기 가능)
- 오늘 마커 표시
- 동기화된 스크롤

### 6.6 폼 뷰
- 전체 크기 데이터 입력 폼
- 항목 간 이전/다음 내비게이션
- 신규 생성 / 기존 편집 모드

### 6.7 차트 뷰
- 바, 라인, 파이, 도넛, 영역 차트
- 그룹핑 필드 + 집계 함수 설정
- 자동 범례/툴팁

### 6.8 저장된 뷰 (Saved Views)
- 필터/정렬/뷰 조합을 북마크로 저장
- 공개 뷰 + 개인 뷰 구분
- 원클릭 적용

## 7. 데이터 CRUD

### 7.1 레코드 작업
- 단일 레코드 생성, 조회, 수정, 삭제
- 벌크 생성 (batch)
- 벌크 수정 (batch update)
- 벌크 삭제 (batch delete)
- 소프트 삭제 (deleted_at, 30일 보관)

### 7.2 레코드 입력 폼
- 모든 필드 타입 지원
- 기본값 자동 채움
- AI 기반 필드 자동 채움 (prefill)
- 항목 복제 (duplicate)
- 항목 인쇄 (print)
- 항목 ID 복사
- 충돌 감지 (동시 편집 시 병합/덮어쓰기 옵션)

### 7.3 조회 파이프라인
- 서버사이드 페이지네이션 (page, limit, offset)
- 정렬 (다중 컬럼, 관계 필드 정렬 → 자동 LEFT JOIN)
- JSON 필터 (AND/OR 중첩, 12+ 연산자)
- 전문 검색 (`?q=`, GIN 인덱스 기반 ILIKE)
- 관계 필드 확장 (`?expand=true`)
- 사용자 필드 확장
- 계산 필드 해석 (formula/lookup/rollup)
- M:N 관계 로딩 (junction 테이블)
- 행 수준 보안(RLS) 적용 (viewer 역할)

### 7.4 벌크 작업
- 다중 행 선택
- 벌크 편집 (BulkEditPanel)
- 벌크 삭제 (확인 필수)
- 선택 건수 표시

- 더블클릭으로 앱 이동

## 9. 프로세스 워크플로

### 9.1 프로세스 설정
- 앱별 워크플로 활성화/비활성화
- 상태(Status) 정의: 이름, 색상, 정렬 순서, 초기 상태 지정
- 전이(Transition) 정의: 출발 → 도착 상태
- 역할 기반 전이 권한 (allowed_roles)
- 사용자 기반 전이 권한 (allowed_user_ids)
- 비주얼 프로세스 플로우 다이어그램

### 9.2 상태 전이
- 현재 상태에서 사용 가능한 전이 목록 조회
- 역할/사용자 권한 기반 필터링
- 전이 시 알림 발송 (생성자 + 지정 승인자)
- 전이 시 자동화 트리거 (`status_change`)

### 9.3 내 작업 (My Tasks)
- 현재 사용자가 전이 가능한 레코드 전체 목록
- 컬렉션별 그룹핑
- 상태 배지 표시
- 앱별 건수 표시

## 10. 자동화

### 10.1 트리거 타입
| 트리거 | 설명 |
|--------|------|
| `record_created` | 레코드 생성 시 |
| `record_updated` | 레코드 수정 시 |
| `record_deleted` | 레코드 삭제 시 |
| `status_change` | 상태 전이 시 (from/to 필터 가능) |
| `schedule` | 크론 스케줄 (타임존 지원) |
| `form_submit` | 폼 제출 시 (폼 슬러그 필터 가능) |

### 10.2 조건
- AND 결합 필드-값 비교
- 연산자: equals, not_equals, contains, gt, lt, is_empty, is_not_empty

### 10.3 액션
| 액션 | 설명 |
|------|------|
| `send_notification` | 인앱 알림 발송 (record_creator / specific_user / field_ref) |
| `update_field` | 레코드 필드 값 업데이트 |
| `call_webhook` | 외부 URL로 POST (JSON, 30초 타임아웃) |

### 10.4 실행 모델
- 비동기 워커풀 실행 (API 응답 비차단)
- 무한루프 방지 (automation_depth 컨텍스트)
- 실행 로그 기록 (성공/에러/건너뜀, 실행 시간)
- AI 기반 자동화 규칙 생성

### 10.5 전역 자동화 페이지
- 전체 컬렉션의 자동화 현황 조회
- 앱별 자동화 페이지 바로가기

## 11. 필터링 & 검색

### 11.1 고급 필터 빌더
- 중첩 조건 (AND/OR 그룹)
- 12+ 연산자: eq, neq, gt, gte, lt, lte, like, in, not_in, is_null, is_not_null, contains
- 필터 칩 (활성 필터 표시)
- 뷰 간 필터 유지
- AI 기반 필터 생성 (자연어 → 필터 JSON)

### 11.2 전문 검색
- `?q=검색어` — text/textarea 필드 전체 ILIKE 검색
- GIN 인덱스 기반 `_tsv` 벡터 검색

### 11.3 정렬
- 다중 컬럼 정렬
- ASC/DESC 전환 (`-` prefix = DESC)
- 관계 필드 정렬 (자동 LEFT JOIN)

## 12. 대시보드 & 차트

### 12.1 앱 대시보드
- 전체 레코드 수 요약 카드
- 숫자 필드 집계 (합계, 평균)
- 선택 필드별 바/파이 차트 자동 생성
- 날짜 필드 타임라인 차트
- 저장된 차트 표시 및 관리
- AI 기반 커스텀 차트 빌더

### 12.2 전역 대시보드
- 전체 컬렉션 개요
- 컬렉션별 레코드 수 표시
- 각 컬렉션 대시보드 바로가기

### 12.3 인터페이스 디자이너
- 커스텀 대시보드 위젯 빌더
- 위젯 타입: 통계 카드, 바/파이/라인 차트, 데이터 테이블, 텍스트 블록
- 위젯 설정: 크기 (1-4 컬럼), 그룹핑, 집계 함수, 표시 필드
- 드래그로 위젯 재배치
- 위젯 삭제
- 인터페이스 레이아웃 저장

### 12.4 데이터 집계 API
- 필드별 합계 조회 (`/totals`)
- GROUP BY 집계 (`/aggregate` — COUNT, SUM, AVG, MIN, MAX)
- 배치 집계 쿼리 (`/aggregate/batch`)

## 18. 접근 제어 & 보안

### 18.1 시스템 역할 (RBAC)
| 역할 | 권한 |
|------|------|
| `director` | 전체 관리 권한, 모든 컬렉션 소유자 수준 |
| `pm` | 스키마 관리, 자동화, 마이그레이션 |
| `engineer` | 데이터 읽기/쓰기 |
| `viewer` | 읽기 전용, RLS 적용 |

### 18.2 컬렉션 수준 접근 제어
- 멤버 기반 접근 (owner/editor/viewer)
- 열린 컬렉션 (멤버 없으면 시스템 역할로 폴백)
- 컬렉션별 권한: entry_view, entry_create, entry_edit, entry_delete

### 18.3 행 수준 보안 (RLS)
- viewer 역할에만 적용
- 모드: none, creator, department, subsidiary, filter(커스텀)
- `buildRLSClause()`가 자동으로 WHERE 조건 추가

### 18.4 보안 미들웨어
- JWT 인증 (쿠키 + 헤더)
- CORS 설정
- API 레이트 리밋 (사용자당 60 req/s, 버스트 120)
- 경로 탐색(path traversal) 차단
- 요청 로깅 (구조화 로그)
- Origin 검증
- `AUTH_DISABLED` 프로덕션 가드 (GO_ENV=production에서 차단)
- Secure 쿠키

## 19. 스키마 마이그레이션

### 19.1 DDL 엔진
- 원자적 트랜잭션 (메타데이터 + DDL 동시 실행)
- up/down DDL 쌍 기록 (롤백 지원)
- 전문 검색 인프라 자동 생성 (_tsv TSVECTOR + GIN 인덱스 + 트리거)
- 부분 인덱스 (deleted_at IS NULL)
- Autonumber 시퀀스 자동 생성

### 19.2 안전 분류
| 등급 | 예시 |
|------|------|
| Safe | 선택적 필드 추가 |
| Cautious | 필수 필드 추가 (기본값 없음), NOT NULL 강화 |
| Dangerous | 필드 타입 변경, 필드/컬렉션 삭제 |

### 19.3 마이그레이션 이력
- 전체 스키마 변경 이력 조회
- 작업 타입별 분류 (create/drop collection, add/alter/drop field)
- 안전 등급 표시
- 롤백 기능
- DDL 문 상세 보기 (펼치기)

## 24. 프론트엔드 인프라

### 24.1 상태 관리
- React Query = 서버 상태 단일 진실 공급원 (Redux/Zustand 없음)
- staleTime: 30초, gcTime: 5분
- 낙관적 업데이트 (onMutate → 스냅샷 → 즉시 반영 → 실패 시 롤백)
- 4xx 재시도 없음, 5xx 2회 재시도

### 24.2 키보드 단축키
- 핫키 도움말 다이얼로그
- 키보드 내비게이션 (그리드, 칸반)

### 24.3 UX 패턴
- 스켈레톤 로딩
- 에러 바운더리 + 에러 상태 (재시도)
- 빈 상태 (도움말 메시지)
- 토스트 알림
- 확인 다이얼로그
- 커맨드 팔레트
- 오프라인 배너
- 언두 토스트 (실행 취소)
- 코치마크 (온보딩 가이드)

### 24.4 모바일 대응
- 앱 빌더/설정/인터페이스 디자이너: 데스크탑 전용 (모바일 리다이렉트)

## 27. 기술 인프라 요약

| 항목 | 구현 |
|------|------|
| 백엔드 | Go 1.26, chi, pgx v5 |
| 프론트엔드 | Vite, React 19, shadcn/ui, Tailwind v4 |

| 실시간 | Server-Sent Events (SSE) |
| AI | vLLM (OpenAI 호환) 연동 |
| 배포 | Docker Compose (Go 싱글 바이너리, embed.FS로 SPA 내장) |
| 알림 | 인앱 + SMTP 이메일 (Slack/Teams 어댑터 확장 가능) |
